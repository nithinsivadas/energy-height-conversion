Following commands have been set up in scc-lite using aliases and the like

`runMatlabScript @functionName` - Run matlab script

`syncGit` - Scynchronize git repository

`backupData` - Backup data in scratch folder into AuroraOptiplex. 

There is a file `exclude-data.txt` which contains data to be excluded in `Tools/Scripts/`

`dataDir` - `"/scratch/nithin/Data"`

`gitRootDir` - `"/usr3/graduate/nithin/git-repos/energy-height-conversion/Tools"`
